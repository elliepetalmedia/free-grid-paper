Create the following files:

2. TECHNICAL REQUIREMENTS
- Tech Stack: HTML5, CSS3, Vanilla JavaScript.
- Library: Use jsPDF via CDN (https://cdnjs.cloudflare.com/ajax/libs/jspdf/2.5.1/jspdf.umd.min.js).
- Persistence: Use `localStorage` to save all slider inputs automatically.
- Constraint: No backend. All PDF generation must happen in the browser.

3. DESIGN SYSTEM (style.css)
- Theme: "Blueprint Dark".
- Colors: Background #0f172a, Sidebar #1e293b, Accent #38bdf8, Text #f1f5f9.
- Layout: 
  - Desktop: Fixed Left Sidebar (320px) for controls. Right area is a flex container holding the Live Preview.
  - Mobile: Column layout (Controls top, Preview bottom).
- Preview Canvas: Center the canvas in the right pane. Give it a white background and a drop shadow to look like paper.

4. GENERATOR LOGIC (script.js)
- Implement `draw()` (renders to Canvas) and `downloadPDF()` (renders to jsPDF).
- Important: The PDF generation must use vector commands (`doc.line`, `doc.circle`), not an image screenshot.
- Tools (Dropdown Select):
  1. Dot Grid: Sliders for Spacing (5-30mm), Size (1-3px), Opacity.
  2. Graph Paper: Sliders for Grid Size (mm), Line Weight, Color (Cyan/Gray/Black).
  3. Lined Paper: Input for Line Height (College=7.1mm, Wide=8.7mm), Margin Toggle (Red line at x=30mm).
  4. Music Staff: Input for Staves per page (8-12). Draw groups of 5 lines + a simple Treble Clef path.
  5. Checklist: Lined paper logic + draw a 5mm square rect at x=15mm.
